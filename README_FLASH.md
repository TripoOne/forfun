# ⚡ CYBER-FLASH // COMMAND CENTER
> **THIS DRIVE IS ARMED WITH PREMIUM LAN TOOLS**

This portable drive contains the **Cyber-Flash LAN Toolset**. It is a zero-install command center designed to provide network diagnostics, secure file sharing, and security monitoring on any machine.

---

## 🚀 Quick Launch

### 🪟 On Windows
1.  Open the **`CyberFlash`** folder.
2.  Double-click **`run_on_windows.bat`**.
3.  Wait for the message: `* PORT 5000 OPENED`.
4.  Open your browser to: **`http://localhost:5000`**

### 🐧 On Linux (Live USB)
1.  Open a terminal in the **`CyberFlash`** directory.
2.  Run: **`bash setup_flash.sh`**
3.  Access the command center at **`http://localhost:5000`**

## 🛠️ Modules Included
- **GHOST_SHARE**: Encrypted P2P file vault.
- **SENTINEL_MODE**: Live security monitor & intruder trap.
- **PULSE_GAUGE**: High-end network speed diagnostics.
- **WAKE_ON_LAN**: Remote power control for network devices.
- **NET_SUITE**: Ping, Traceroute, Port Scanner, and LAN Sweeper.
- **NEON_CORE**: 4 Interchangeable visual themes.
